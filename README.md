# 📱 Desafio DIO Java - iPhone com Interfaces

Projeto em Java que simula um iPhone com três funcionalidades principais, usando interfaces e uma classe concreta.

---

## 💡 Funcionalidades

- **Reprodutor Musical**
  - selecionarMusica
  - tocarMusica
  - pausarMusica

- **Aparelho Telefônico**
  - ligar
  - atenderLigacao
  - iniciarCorreioVoz

- **Navegador de Internet**
  - exibirPagina
  - adicionarNovaAba
  - atualizarPagina
